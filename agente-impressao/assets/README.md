# Ícones necessários

Coloque aqui os ícones do aplicativo:

- `icon.png` — Ícone principal do app (256x256px PNG)
- `icon.ico` — Ícone para o instalador Windows (256x256px ICO)
- `tray.png` — Ícone da bandeja do sistema (16x16px ou 32x32px PNG)

## Como gerar o .ico

1. Crie um PNG 256x256 com o logo do MarketLajinha
2. Use https://convertio.co/png-ico/ para converter para .ico
3. Salve como `icon.ico` nesta pasta
